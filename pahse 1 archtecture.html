Phase 1 (SAS + Cloud VM Only)

In Phase 1, we move SAS to a Cloud VM and run a monthly pipeline with automated QA checks. This supports growth, heavy joins, and multi-user access without adding Snowflake yet.

What Phase 1 Delivers

  • Monthly pipeline: ingest → transform → validate → publish
  • Automated QA: ~35 test cases (counts, duplicates, missingness, code checks)
  • Repeatability: standard outputs + run logs
  • Collaboration: shared compute environment + access control

Key Components

  • Cloud VM: SAS 9.4 batch + schedulers
  • Staging folder: mounted storage / VM disk
  • Validation outputs: Excel/CSV + curated SAS datasets
  • Alerts: optional email on failures
