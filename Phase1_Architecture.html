Phase 1 (SAS + Cloud VM Only)

A scalable foundation for monthly Medicaid ingestion and automated QA test cases, using SAS running on a Cloud VM with scheduled jobs and standardized outputs.

What Phase 1 Delivers

  • Monthly pipeline: load → transform → validate → publish outputs
  • Automated QA: ~35 test cases (counts, duplicates, missingness, code checks, line logic)
  • Repeatability: run logs + versioned outputs
  • Shareable results: Excel/CSV reports + summary tables

Why Cloud VM

  • Scales: upgrade RAM/CPU for heavy joins
  • Predictable: consistent runtime vs N-drive variability
  • Multi-user ready: shared environment & access control
  • Pay-as-you-go: run only when needed

How Automation Runs (Monthly)

Start VM → ingest → build tables → run QA → publish reports → stop VM.

Monthly Run Steps

  • 1) Start VM
  • 2) Ingest files into staging
  • 3) Build analytic tables
  • 4) Run QA tests and capture pass/fail
  • 5) Publish outputs for stakeholders
  • 6) Stop VM to control cost

Outputs (What People See)

  • Validation summary table (metrics + timestamps)
  • Excel QA report (failures first)
  • Curated datasets for downstream analytics
  • Run logs for audit trail
